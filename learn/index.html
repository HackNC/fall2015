Web Development

Web development (or web dev) refers to the tools used to develop a website to put on the web. This includes HTML, CSS, JavaScript, jQuery, Ruby, and much more. Web dev makes the Internet fun, so take a look at the following links to become a web guru!

Learn to Code HTML and CSS

This is a good tutorial for learning HTML and CSS. Aimed towards beginers, Learn to Code HTML and CSS takes you step by step through the basics.

JavaScript Tutorial

This basic JavaScript tutorial goes through the basics of JavaScript and covers everything from syntax to debugging.

Mozilla Developer Network

The Mozilla Developer Network (MDN) is one of the most popular resources for web developer on the web. It is a network of easily readable documentation on HTML, CSS, JavaScript, graphics, API's and more. The site also includes tutorials on HTML, CSS, and JavaScript.

Virtual Reality

Virtual reality, besides being very cool, has many awesome applications. Besides being used in gaming, virtual reality is used in the military, healthcare, education, sports, and many other areas. Take a look at the following links to see of the cool things going on with virtual reality!

6 Ways Virtual Reality is Already Changing the World

This article includes videos on six cool ways that virtual reality is impacting the world, including treating children with autism and allowing surgeons to practice techniques before trying them out on patients.

What is Virtual Reality?

The virtual reality site has many useful links, but this page explains what virtual reality is pretty well, and the tools that can be used to create the experience.

10 Mind-Blowing Oculus Rift Experiments That Reveal VR's Practical Potential

This article presents ten very cool projects done with the Oculus Rift, a virtual reality head-mounted display. These projects include using the Rift to give medical students a first person view of surgery, and how NASA is working on using the Rift to move robots in space.

Android App Development

Android app development uses Java, so those of you who know it have a head start! Even if you don't know Java, making an android app can be fun and easy, and in the end you will have an awesome working app! Here are tutorials on Java and Android development. Although not at popular as it's competitor, iOS, it does provide access to most functionalities on the device.

Java Tutorial

This is a simple and quick tutorial that can help you learn Java. It starts with setting up your environment, and if you want to learn more, there is the option to learn more advanced things such as networking.

Building Your First App

Android put out their own tutorial on how to build Android apps, and they go through the works, starting with setting up your workspace and going through building a UI.

I Want to Write Android Apps. Where Do I Start?

This article gives the different ways to develop Android apps, and includes resources for developers. A great resource to check out!

iOS App Development

Want your apps on the Apple App Store? Look no further than iOS development. iOS apps are created using Swift, which recently became open source, or Objective-C. The Apple App Store leaves more money for its developers, so learn to code for the most used smartphone and possibly a greater profit!

Start Developing iOS Apps Today

Apple put out their own tutorial on how to build iOS apps, and it starts with downloading the tools that you need to build, and then leads you through building your own note app.

Objective-C Tutorial

This tutorial is good to get a grasp on the language. It covers the basics and offers advanced options for those who want to learn more.

The Swift Programming Language

This quick tutorial, from Apple, quickly goes over the basics of Swift and includes a language reference.
